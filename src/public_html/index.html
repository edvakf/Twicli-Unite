twicli

twicli is a browser based Twitter client.

  • No need for installation. All you need is a Web browser.
  • Automatically fetches timeline, and show with cool animation.
  • Use cross domain JavaScript to access Twitter directly (no server needed), so your password is sent to Twitter only.

Maybe convenient when you are in an Internet cafe, doing something else while viewing Twitter timeline.
Works on Firefox, Safari, Opera, IE8 at the moment, as well as Safari on iPhone/iPod touch
Works on IE7 too, but some part aren't displayed properly.

Usage

Launch

Just click the link above, or run it as a bookmarklet.
If you get asked for authorization, check if the origin of authorization is really "twitter.com", then enter your Twitter account/password.

Be careful not to hit API request limit (150 requests per hour) by clicking too much ^^;

How to Use

Customize

User Stylesheet

You can add your own CSS in "user stylesheet" section of the "+" tab. For example;
  • to hide user icons : .uicon { display: none; }
  • wider lines : .status { line-height: 1.5; }

Plugins

You can register plugins by setting them at the "Preferences" in the "+" tab.
Currently, there are plugins below:
  • regexp.js : Add a tab that extracts timeline that has specific keywords, ID, etc. Detail(in Japanese)
  • outputz.js : Count how many letters you tweet, and send it to Outputz. Not registedred as default.
  • search.js : Add keyword search by Twitter Search to the "+" tab.
  • search2.js : Add keyword search by twitter search(yats) to the "+" tab. Not registedred as default.
  • favotter.js : Indicate latest popular favorited tweets like [fav:3]. Thanks to Favotter. Not registedred as default.
  • followers.js : Colorize tweets by users you don't follow in blue. To use this, you need to fetch the list of followers by clicking the "Renew" button at the "Color followers" section. Not registedred as default.
  • sound.js : Play sound when updating timeline/reply. Not registedred as default.

About Opera Unite version

twicli unite, the Opera Unite version of twicli, is exactly the same as twicli except for some additional plugins. It can be launched by the above bookmarklet, or directly open this.
You can also use it in Opera's panel.

Plugins

  • unite_cache.js : Cache timeline so that the page loads faster.
  • unite_longer_url.js : Expand short urls.
  • unite_favotter.js : Almost identical to the original favotter.js. (Not enabled by default)
